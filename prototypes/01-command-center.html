Modelo mental

BMAD não é modelo de IA, IDE ou garantia automática de qualidade. É sistema modular que transforma intenção em trabalho verificável. Abra novo chat para cada workflow: arquivos preservam decisões; contexto limpo reduz ruído.

Escolha trilha
