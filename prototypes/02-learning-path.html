Primeiro passo

Instale. Pergunte. Caminhe.

npx bmad-method install

Depois: abra sua ferramenta de IA no projeto e peça orientação. Neste guia, o exemplo é $bmad-help o que faço primeiro?

Node.js 20.12+ necessário. Git recomendado. Instalação cria _bmad/; entregáveis vão para _bmad-output/. Forma de chamar skills varia conforme ferramenta.

Entender como funciona

Pessoa define intenção. Workflow conduz processo. Agente aplica especialidade. Artefato guarda decisão.

Começar rápido

Trilha principal

Quatro fases. Uma linha contínua.

Fluxo canônico do track BMad Method. Brief é opcional; PRD, arquitetura e prontidão funcionam como gates antes do código.

01

Análise

Opcional

Descubra problema, público, mercado e viabilidade antes de fechar solução.

Entra curiosidade · sai visão$bmad-product-brief
ideia›pesquisa›brief
02

Planejamento

Gate: PRD

Defina o que construir, para quem e como medir sucesso. Se interface for central, detalhe UX.

Entra visão · saem requisitos$bmad-prd $bmad-ux
brief›PRD›UX
03

Solutioning

Obrigatória

Fixe invariantes técnicas, transforme requisitos em épicos e valide alinhamento antes do sprint.

arquitetura›épicos›readiness
